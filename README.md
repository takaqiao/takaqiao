## 你好 / Hi

我是 **Taka**，跑团 GM，目前在 Foundry VTT v14 上跑《憎恶的金库》(*Abomination Vaults*)。GitHub 上的开源活动主要围绕：

- 给 Foundry VTT 写实用的小模组（PF2e / SF2e / Crucible 体系）
- 维护中文翻译合集（PF2e / SF2e / Crucible / Ember 等）
- 翻译流水线工具（Babele 源文件提取、批量校对、自动翻译）

I'm Taka — a Pathfinder 2e GM running games on Foundry VTT v14. Most of what I publish here is community work for the Chinese-speaking PF2e / SF2e / Crucible VTT scene: translation modules and small quality-of-life add-ons.

## 精选项目 / Featured

### Foundry VTT 模组 / Modules

| Module | 简介 / What it does |
|---|---|
| **[pf2e-dsn-persistent-bridge](https://github.com/takaqiao/pf2e-dsn-persistent-bridge)** | 把 Dice So Nice 画布上的持久骰子作为 PF2e 检定 / 伤害 / 重投对话框的输入。<br>Use the dice you physically throw on the canvas (DSN persistent dice) as input for PF2e roll dialogs. |
| **[pf2e-xp-tool](https://github.com/takaqiao/pf2e-xp-tool)** | PF2E XP 预算工具：精英 / 弱小模板预览与多策略补差值方案。<br>Encounter XP planner with elite/weak preview and multi-strategy gap fillers. |
